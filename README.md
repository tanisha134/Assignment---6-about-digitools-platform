__DigiTools Platform__

This is a simple frontend project where I built a digital tool website using React. The main goal of this project is to practice component-based design, responsive layout, and basic state management.

__What This Project Is About__

In this project, I created a website where users can see different digital tools, pricing plans, and basic information about platform. it is not connected to any backend yet, but is shows how a real website UI can look and work.

__Features__

* A responsive Navbar with a cart icon and links
* A Banner Part 
* A Stats Part
* A Products part (with add to cart system)
* A steps part (how to get stated)
* A pricing part
* A footer with links and social icons
* Toast notifications when interacting with products

__Technologies I Used__

* React (For building UI)
* Vite (For fast development)
* Tailwind CSS (For styling)
* DaisyUl (for some UI components)
* React Icons (for icons)
* React Toastify(for notifications)  

__How I Built It__

First I created the project using Vite. Then I installed Tailwind CSS and DaisyUI for styling. After that, I created different components like Navbar, Banner, Products, etc.

Each part of the website is made as a separate component. I used useState to manage the cart data and passed it through props.
